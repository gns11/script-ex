산술연산자
+ => 덧셈
- => 뺄셈
* => 곱셈
/ => 나누기
% => 나머지
** => 제곱
-- => 감소연산자
++ => 증가연산자

결합연산자
+ => 결합/연결 연산자

대입연산자
= => 우항을 좌항으로 대입
+= , -= , *= , /= , %= => 산술 대입연산자

비교연산자
a>b => 초과
a<b => 미만
a<=b => 이상
a>=b => 이하
a||b => or
a&&b => and
a!=b => 부정

3순위 : 산술연산자(*,/.+,-)
4순위 : 비교연산자 
5순위 : 논리연산자
6순위 : 대입연산자
